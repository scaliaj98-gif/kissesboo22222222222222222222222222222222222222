# SnapRecord Pro v2.0

A complete, Loom-style screen recording + screenshot Chrome extension.

## Features
- 🎬 Screen/tab/webcam recording with PiP
- 📸 Screenshots: visible, region, full-page
- ✏️ Screenshot annotation editor (pen, highlighter, arrows, shapes, text, blur, emoji)
- 🎞️ Video editor with timeline, trim, effects, text overlays
- 🤖 AI Assistant sidebar (OpenAI + Gemini support)
- 🗂️ Local library with search and filtering
- ⌨️ Keyboard shortcuts
- 💬 Video bubbles on Gmail, Slack, Notion, GitHub

## Load Unpacked
1. Open `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select this folder

## Keyboard Shortcuts
- `Alt+Shift+V` — Capture visible area
- `Alt+Shift+F` — Capture full page
- `Alt+Shift+R` — Select region
- `Alt+Shift+S` — Start/stop recording
- `Alt+Shift+A` — Open AI Assistant

## AI Setup
Go to Settings and add your OpenAI (`sk-...`) or Gemini API key for full AI assistance.
